Para completar a instalação do site, adicione os seguintes arquivos de imagem nesta pasta:

1. logo.png - Logo da EB Cursos (preferencialmente em fundo transparente)
2. curso-ia.jpg - Imagem para o curso de IA
3. curso-engajamento.jpg - Imagem para o curso de Engajamento

As imagens devem ter as seguintes dimensões recomendadas:
- Logo: 200x50 pixels (mantenha a proporção original)
- Imagens dos cursos: 800x450 pixels
